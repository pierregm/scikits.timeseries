.. currentmodule:: scikits.timeseries

========
Plotting
========


The :mod:`lib.plotlib` module simplifies the production of time series plots 
using matplotlib_.
It relieves the user from the burden of having to setup appropriately spaced 
and formatted tick labels.

It is recommended to have some experience with matplotlib_ before using 
the :mod:`~scikits.timeseries.lib.plotlib` or following the examples below.
A `tutorial <http://matplotlib.sourceforge.net/users/index.html>`_ is available 
online.


.. _matplotlib: http://matplotlib.sourceforge.net
